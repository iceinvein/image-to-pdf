# Image to PDF

This is a simple app that allows you to convert an image to a PDF file. A no nonsense app that does everything locally on your device.

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ios

   ```bash
    npx expo run:ios
   ```

   android

   ```bash
    npx expo run:android
   ```
